# Metric-Imperial-Converter
A Python Tkinter GUI application that converts distance, weight, and temperature units. Features a clean multi-window interface, clear user prompts, and formatted results, demonstrating GUI design, event-driven programming, and unit conversion logic.

# Tkinter Unit Converter

## Description
Tkinter Unit Converter is a Python GUI application that performs common
measurement conversions using an interactive interface.

The application allows users to convert:
- Miles ↔ Kilometres
- Kilograms ↔ Grams
- Celsius ↔ Fahrenheit

Each conversion type opens in its own window and provides clear prompts,
input fields, and formatted results.

## Purpose
This was a personal project which was created as a learning exercise to practice:
- Building graphical user interfaces with Tkinter
- Handling user input and events
- Applying mathematical logic for unit conversions
- Structuring a multi-window Python application


## Features
- Simple and intuitive Tkinter GUI
- Multiple conversion categories
- Real-time calculation with formatted output
- Clear and reset functionality

## Technologies Used
- Python 3
- Tkinter

## How to Run
1. Make sure Python 3 is installed.
2. Run the program:
   In Python IDLE
   python tkinter_unit_converter.py
